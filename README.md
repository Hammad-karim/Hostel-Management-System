 # Hostel Management System (SDA)

This project was developed as part of the Software Design & Analysis (SDA) course. 
It focuses on requirement analysis and system design for a Hostel Management System.

## Project Overview
The system is designed to manage hostel operations such as student registration, room allocation, and record management. The project emphasizes structured analysis and proper software design practices.

## Features
- Student and hostel record management
- Room allocation process
- Basic fee tracking (conceptual)
- Well-defined system workflows
